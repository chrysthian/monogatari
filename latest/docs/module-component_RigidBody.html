Module: component/RigidBody

Component used for interaction with the box2d.js Emscripten port.

Box2D has been tuned to work well with moving objects between 0.1 and 10 meters. So this means objects between soup cans and buses in size should work well. Static objects may be up to 50 meters big without too much trouble. Box2D is tuned for meters, kilograms, and seconds.
Parameters:
conversionFactor Number Multiplies the position from physics world (meters) to screen coordinates (pixels). Defaults to 1 (probably not what you expect).
bodyDef b2BodyDef Box2D physics body definition
materialDef b2FixtureDef Box2D physics material definition

Members

(static) TYPE :Number

Enumeration of rigid body types.
Properties:
STATIC Number A static body is a body which isn’t affected by world forces it does not react to collisions. It can’t be moved.
KINEMATIC Number A kinematic body is an hybrid body which is not affected by forces and collisions like a static body but can moved with a linear velocity like a dynamic body.
DYNAMIC Number A dynamic body is a body which is affected by world forces and react to collisions.
