collection/Tree~ Tree

new Tree()

Collection of data elements sorted hierarchically from root to leaf, with child nodes connected by means of a pointer. Elements can easily be inserted or removed without reallocation or reorganization of the entire structure because the data items need not be stored contiguously in memory.
