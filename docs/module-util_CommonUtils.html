(static) parseUnitSizeToPixel(text) → {Number}

Parses a given String to the float pixel value of it.
Parameters:
text String
Returns:
Number
 
Example:
parseUnitSizeToPixel('10pt'); //12.5
parseUnitSizeToPixel('10pc'); //150
parseUnitSizeToPixel('10mm'); //35.43307
parseUnitSizeToPixel('10cm'); //354.3307
parseUnitSizeToPixel('10in'); //900

(static) store(key, val, isObject)

Stores a given value on the Local Storage.
Parameters:
key String Unique identifier on the Local Storage
val Object The value to be stored
isObject Boolean Boolean to flag if is an Object
Example:
store('num', '1');
store('on', 'true');
store('name', 'pamela');
store('obj', {'hello': 'world'}, true);
