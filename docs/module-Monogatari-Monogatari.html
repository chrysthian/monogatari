Methods

init(bgcolor, widthopt, heightopt, targetopt)

Engine initialization function. Creates the default Scene and Camera and register the input events.
Parameters:
bgcolor String Hexadecimal background color
width Number <optional>
 Width of the canvas in pixels. Defaults to screen resolution.
height Number <optional>
 Height of the canvas in pixels. Defaults to screen resolution.
target DOMElement <optional>
 Target node of the Dom tree to create a canvas renderer. It is attached to the body if not provided.

render()

Engine render function. Cycles through all Cameras and Scenes rendering the registered components on screen.

run()

Engine main heartbeat function.

update()

Engine logical update function. Control engine timer and cycles through all GameObjects and needed components updating them.
