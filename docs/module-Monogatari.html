(static) init(bgcolor, widthopt, heightopt, targetopt)

Engine initialization function. Creates the default Scene and Camera and register the input events.
Parameters:
bgcolor String Hexadecimal background color
width Number <optional>
 Width of the canvas in pixels. Defaults to screen resolution.
height Number <optional>
 Height of the canvas in pixels. Defaults to screen resolution.
target DOMElement <optional>
 T- arget node of the Dom tree to create a canvas renderer. It is attached to the body if not provided.

(abstract, static) loadingProgress(percentage)

Function to be overridden with a way to show the asset loading progress.
Parameters:
percentage Number The total percentage of loaded assets

(static) render()

Engine render function. Cycles through all Cameras and Scenes rendering the registered components on screen.

(static) run(loadedopt)

Engine main heartbeat function.
Parameters:
loaded Boolean <optional>
 If all assets in current world finished loading

(static) update()

Engine logical update function. Controls engine timer and cycles through all GameObjects and needed components while updating them.
