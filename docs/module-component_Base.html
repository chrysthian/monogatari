Module: component/Base

Base class that all components extend.

Members

(static) STATE :number

Enumeration of component states.
Properties:
INITIALIZING number Component was created and may need further steps before use
BUFFERING number Component is buffering resources
LOADED number Component resources are loaded
READY number Component is built and ready for use
RUNNING number Component is active
REGISTERED number Component is registered for internal management
FAILED number An error occurred and the component cannot be used

(static) TYPE :number

Enumeration of component types.
Properties:
SPRITE number Managed 2D asset renderable by THREE.
RIGID_BODY number Physics for Box2d.
AUDIO number Audio source.
