Class: GameObject

core/GameObject~ GameObject

new GameObject(idopt, updateopt, positionopt, rotationopt, scaleopt)

The main build block of the engine. Create the game classes by inheriting from the Game Object. For quick creation of GOs, it is also possible to use composition.
Parameters:
id String <optional>
 Identifier. Not mandatory, but recommended
update function <optional>
 The object logic. Not necessary if using inheritance
position THREE.Vector3 <optional>
 Initial position
rotation THREE.Vector3 <optional>
 Initial rotation
scale THREE.Vector3 <optional>
 Initial scale
Example:
// Composition
var myGO = new m.GameObject( myGOId, myGOUpdateFunction );

// Inheritance
var MyGOClass = function( id ) {
 m.GameObject.call( this, id );
};
MyGOClass.prototype = Object.create( m.GameObject.prototype );
MyGOClass.prototype.update = function() { ... };
