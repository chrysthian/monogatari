Module: component/Sprite

Sprite and sprite sheet component, rendered by three.js.
Parameters:
source String <optional>
 Sprite sheet URI
width Number <optional>
 Width in pixels for one animation frame
height Number <optional>
 Height in pixels for one animation frame
rows String <optional>
 Number of rows on the sprite sheet
cols String <optional>
 Number of cols of the sprite sheet

Extends

  • module:component/Base

Members

Current column of the sprite sheet, ranges from 0 to N (like an array).
Default Value:
  • 0

cols :Number

Number of columns on the sprite sheet, ranges from 1 to N (like array.length).
Default Value:
  • 1

frame :Number

Current animation frame of the sprite sheet, ranges from 1 to N (like array.length).
Default Value:
  • 1

Methods

buildMesh()

Builds the THREE.Mesh with current geometry and material.

hide()

Flag the THREE.Material to NOT be rendered.

nextFrame()

Moves to the next animation frame of the sprite sheet.

rotateAroundObjectAxis(axisopt, radiansopt)

Rotate an object around an arbitrary axis in object space (reference).
Parameters:
axis THREE.Vector3 <optional>
 The reference axis for rotation
radians Number <optional>
 Radian degrees to rotate

rotateAroundWorldAxis(axisopt, radiansopt)

Rotate an object around an arbitrary axis in world space (reference).
Parameters:
axis THREE.Vector3 <optional>
 The reference axis for rotation
radians Number <optional>
 Radian degrees to rotate
Example:
// rotation of 90 degrees on the x-axis
var xAxis = new THREE.Vector3(1,0,0);
rotateAroundWorldAxis( xAxis, Math.PI / 180);

setFrame(frame)

Set the animation frame to the given frame number. If a number greater than the number of frames in the sprite sheet is informed, the algorithm cycles through the frames as many times as needed
Parameters:
frame Number Frame number

show()

Flag the THREE.Material to be rendered.

updateMesh()

Update the internal THREE.Mesh with the THREE.Material and THREE.Geometry.
